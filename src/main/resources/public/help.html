Dragonnet CSL help

Central Login System

What is DragonProxy ?

DragonProxy is a software placed between a Minecraft Bedrock client and a Java Minecraft server.

You can read more on the Github page here or the SpigotMC page here.

Why entering my Mojang credentials ?

You have been requested to enter your Mojang account name and password to link it to your XBoxLive account.

You must link your accounts if you want to join online mode Java Minecraft servers.

You can disable password saving in our databases by not checking the form "save my password" checkbox, that will deliver a "one time use" Mojang token, if the token is invalidated by logging on another place, you will be asked to do another time the operation.

How it works ?

1 - On your first connecting on a CLS enabled DragonProxy server, your Xbox Live Minecraft public profile is stored in our server, a PIN code is generated, it's valid for 5 minutes.

2 - You are asked to enter this PIN code on the register page.

3 - If the PIN code is valid, the register page will display your Xbox Live Minecraft display name

4 - You need now to enter your Minecraft username and password, then choose if you want to save it on the CLS database.

5 - The CLS server will login your account to generate a new access token.

6 - You can login again on DragonProxy with you Minecraft (W10/PE) account, it will contact CLS server to authenticate you on the targeted Java Miencraft server.

Each time you login on a CLS enabled DragonProxy server, your token will be checked like the Minecraft launcher does, it's the same process.

Is my Mojang password secured ?

Paswords saved in our database are ciphered with a complex character chain, and the database itself is secured by multiple layers, stored in a safe place.

At the time of writing this, I consider storing password is unsafe as the engine is in early alpha, untested in production.

Can I unlink my accounts ?

Yes of course, when linking, we have displayed you an "unregister link", just click it and we instantaneously forget your Mojang account datas, including tokens.

If you don't have the unlink account link, You can also unlink your account by logging your Mojang Minecraft account on the unlink page.

You can also connect on a a CLS enabled DragonProxy server and type "/cls unlink".
